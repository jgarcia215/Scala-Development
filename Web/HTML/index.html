About Scala

Scala is a programming language that was created by Martin Odersky and first released in 2003. Its development was influenced by several programming languages and paradigms, and it aimed to provide a language that combines object-oriented and functional programming features. Odersky developed Scala at the École Polytechnique Fédérale de Lausanne (EPFL) in Switzerland in 2001. Early adopters of Scala are tech companies such as Twitter and Linkedln, which used Scala for back end services. In the early 2010's it went through the standardization process and thus created the Scala Standard Library. Odersky helped develop the precursor to the Java compiler which motivated him to create Scala -- which was designed to address the criticisms of Java.

Sample Programs

Though Scala parameters may look odd, it does follow OOP principles familar to those who have used Java.

// Define a simple class
class Person(name: String, age: Int) {
  def displayInfo(): Unit = {
   println(s"Name: $name, Age: $age")
  }
}

// Create an instance of the class
val person1 = new Person("Alice", 25)

// Call a method on the object
person1.displayInfo()


Scala can perform basic functions like Java, such as using a for loop. However, it also offers shorter and cleaner versions of these tools as well.

// Create a list of numbers
val numbers = List(1, 2, 3, 4, 5)

// Loop through the list and print each element
for (num <- numbers) {
  println(num)
}

// Using a for each loop
numbers.foreach(num => println(num))


About Project

This website is our submission for the Language Website assignment of CS354. On the backend, three Scala programs are used, allowing for the site to scrape Yahoo Finance for stock information, parse the information gleaned, and export the information as a graph. On the frontend, HTML and JavaScript are used to create the site you're looking at, taking user input via dropdown menus, and using it to generate a graph with the Scala program.

Our project was chosen in order for us to gain experience not only with Scala, but with web development as well. None of us had experience with webservers prior to beginning this project, so choosing a project that required additional interactions with websites, such as Yahoo Finance, proved to be an educational experience.
